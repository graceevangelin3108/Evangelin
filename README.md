# Google Search Clone

A simple **Google-inspired Search Page** built using **HTML, CSS, and JavaScript**. This project is created for learning purposes to understand the basics of front-end web development.

## Features

* Responsive layout
* Google-style logo
* Search input box
* Google Search button
* "I'm Feeling Lucky" button
* Search using the **Enter** key
* Clean and simple user interface

## Project Structure

```
Google-Search-Clone/
│── index.html
│── style.css
│── script.js
└── README.md
```

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.
4. Enter a search term and click **Google Search** or press **Enter**.

## Files

### index.html

Contains the structure of the webpage, including the logo, search box, and buttons.

### style.css

Contains all the styling for the webpage, including layout, colors, fonts, and button effects.

### script.js

Handles:

* Google Search
* "I'm Feeling Lucky" feature
* Searching when the Enter key is pressed

## Note

This project is a **Google-inspired clone** created **only for educational purposes**. It is not affiliated with or endorsed by Google. It uses Google's public search URL to perform searches.
